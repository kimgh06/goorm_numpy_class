# 스페셜 메소드
스페셜 메소드가 충격인 이유 뿌슝빠슝

```python
a, b = 10, 20
```

zip() 같은 것 끼리 맞춰줌

오브젝트는 오브젝트와의 계산이 가능함.

```python
a+b 
a.__add__(b)
```

서로 같지만 +기호를 메소드로 인터프리터가 변환함.

```python
len(a)
a.__len__()
```
len을 아래에 있는 메소드로 인터프리터가 바꿔준다.
