# 매트릭스에서 인덱싱

![Alt text](image-38.png)

```python
import numpy as np

a = np.arange(9).reshape((3,3))

a[0, 0] # 0
a[2, 2] # 8

a[0, 1:] # 1 2
a[1:, 0] # 0 3 6
a[1:, 1:] # [[4 5] [7 8]]

a[:, ::-1] # [[2 1 0] [5 4 3] [8 7 6]]

a[::-1, ::-1] # [[8 7 6] [5 4 3] [2 1 0]]

a[0, ...] # 뒤에 따르는 모든 차원을 가져오세요

a[..., 1] # 마지막 차원 앞에 따르는 모든 차원을 가져오세요
```
